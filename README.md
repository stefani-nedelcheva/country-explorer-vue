# Country Explorer

A Vue 3 single-page app that fetches country data from the REST Countries API and presents it in a searchable, filterable interface.

## Features

- Search by country name, official name, or capital
- Filter by region
- Detail panel with flag, population, timezones, languages, and border countries
- Vue 3 Composition API
- Tailwind CSS styling

## Stack

- Vue 3
- Vite
- Tailwind CSS
- REST Countries API

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
